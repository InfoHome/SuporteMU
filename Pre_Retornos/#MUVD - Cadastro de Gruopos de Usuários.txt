Na tela <b>Cadastro -> Grupos</b> siga as orientações abaixo:

1 - Informe o nome do grupo no campo <b>Grupo</b>
2 - Informe o sistema <b>Faturamento</b> no campo <b>Sistema</b>
3 - Marque o campo <b>Acesso Permitido</b>
4 - Se não existir cadastro para esse Grupo ==> Sistema, clique em novo para cadastrar esse sistema para o grupo desejado.

5 - A opção que deseja liberar está no Menu <b>Relatórios</b> Acesso: <b>Reimpressão de Danfe</b>
