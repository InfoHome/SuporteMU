<b>OK, Vamos lá!</b>

1 - Vamos fazer alguns levantamentos para atualização:

1.1 - Vou considerar que a versão atual do meu validar seja <b>20800</b>
- Acesso o caixa e verifique qual o sua.

1.2 - Vou considerar que a que o banco de dados que utilizo seja da Microsoft <b>SQL SERVER</b>
- Verifique qual o seu, em caso de dúvidas verifique com a equipe técnica de TI.

1.3 - Vou considerar que a versão atual do meu PAF ECF seja <b>114</b>

<i><b>Dica!</b>
Para descobrir a versão do seu PAF-ECF podemos seguir essa lógica:

1.3.1 - Acesse: Financeiro Contábil[Masteren] / Cadastros / Núcleo Comum / Filial, Clique no Botão "OK"
- Selecione a filial que está atualizado e dê duplo clique para abrir seu cadastro.

1.3.2 - No menu "Guias" da Filial escolhida, clique sobre a guia "Parâmetros"
1.3.3 - No campo "Domínio" selecione: "Sistema de Vendas" e clique na varinha de busca a frente do campo.
1.3.4 - No final dos parâmetros localize o parâmetro: <b>Versão PAF/ECF</b>
1.3.5 - Veja qual o valor desse parâmetros: Vazio, 110, 113 ou 114.

Atenção: Se for Vazio indica que também é 110, porém para confirmar acesse: C:\Microuni\Enterprise\
- Localize o programa <b>ESGE-ECF.EXE</b>
- Clique com o botão direito do Mouse sobre ele e clique em propriedades
- Clique na aba Detalhes
- Localize a Descrição: "Versão do Arquivo", nesta informação deve estar o Valor 110.0.1.0, ou seja, <b>110</b>
</i>

2 - Para atualizar o Item 1.1 acesse: http://download.microuniverso.com.br/TradSql/V.Unificada%20especifica/ESGE/ECF%20HCU/?C=N;O=D
2.1 - Baixe a última versão disponível do sistema, hoje neste momento 10/03/2017 09:51 a versão atual é <b>Validar25700.zip</b>
2.2 - Após baixar você vai descompactar com a senha <B>BLZA</B>
2.3 - Após descompactar você irá renomear para <b>Valida114.exe</b> para atender o item 1.3 acima.

Atenção.: 
- No item 1.3, se a versão do PAF-ECF que você utiliza for <b>110</b>, você irá renomear o arquivo descompactado para <b>Validar.exe</b>
- No item 1.3, se a versão do PAF-ECF que você utiliza for <b>113</b>, você irá renomear o arquivo descompactado para <b>Valida113.exe</b>
- No item 1.3, se a versão do PAF-ECF que você utiliza for <b>114</b>, você irá renomear o arquivo descompactado para <b>Valida114.exe</b>

2.4 Após renomear o arquivo para nomenclatura correspondente a sua versão do PAF-ECF instalada no sistema atualmente, você 
irá copiar o <b>Validar.exe</b> ou o <b>Valida113.exe</b> ou <b>Valida114.exe</b> para o diretório: <b>C:\Windows\AppTemp\</b>

==> Pronto! Nesse ponto sua versão do executável está atualizado!

3 - Agora Vamos atualizar o banco de dados.
---------------------------------------------------------------
3.1 - Para atualizar o banco de dados você precisa saber qual banco de dados está trabalhando: Oracle ou SQL.
3.2 - Sabendo disso acesse: http://download.microuniverso.com.br/TradSql/V.Unificada%20especifica/ESGE/Scripts/?C=N;O=A
3.3 - Se for Oracle, você irá utilizar os scripts que comecem com <b>ORA_</b>
3.4 - Se for SQL, você irá utilizar os scripts que comecem com <b>SQL_</b>
3.5 - Então para continuar com este exemplo, para atender o item 1.3 vou seguir pelo item 3.4, Banco de Dados "SQL".
